# 口语评分三个方向
- Task1 内容 -- 首条理由充分原则 -- 实词汇-- timing （鸡精准备）
- Task2 模版 -- 两个基本点 -- 信息筛选原则（分类）--timing （同意 不同意 不完全同意）（连接词说错了一定要回来改）
- Task3 & Task4 即时长难句 -- 信息筛选原则（分类） --timing



# 复习方案
- 基本功：影子跟读 / 每天30分钟+
- 独立口语：时刻关注鸡精/自写提纲 + 表达方式 限时练习 + 录音自评 内部交流 自写答案 固定用法
- 综合口语：秒一 保二，取舍三四 集中优势兵力原则 
- 精读/精听 + 录音自评
